┌─────────────────────────────────────────────────────────────────┐
│ ⚡ QUICK DEPLOY (3 MINUTES)                                   │
└─────────────────────────────────────────────────────────────────┘

1️⃣ Buka Vercel → Add New Project → Import from GitHub

2️⃣ Pilih repo: masudrahmatulah/balikin

3️⃣ Add Environment Variables (copy dari VERCEL_ENV_SETUP.txt):
   - DATABASE_URL
   - BETTER_AUTH_SECRET
   - BETTER_AUTH_URL
   - NEXT_PUBLIC_BETTER_AUTH_URL
   - RESEND_API_KEY
   - FONNTE_API_TOKEN
   - dll (lihat file lengkap)

4️⃣ Click DEPLOY

5️⃣ Selesai! 🎉
