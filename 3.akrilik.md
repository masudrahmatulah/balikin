Layanan **Akrilik Premium (Ultimate)** adalah "karya seni" dari bisnis Anda. Ini bukan sekadar alat pelacak, melainkan aksesori personal yang memiliki nilai emosional tinggi. Karena harganya paling mahal, fiturnya harus terasa sangat **eksklusif** dan **aman**.

Berikut adalah panduan lengkap untuk kasta tertinggi dari produk **Balikin**:

---

## 1. Fitur Eksklusif (The Ultimate Benefits)

Apa yang membuat orang mau bayar lebih untuk Akrilik? Selain fisiknya yang keren, fitur digitalnya harus "Dewa":

* **Real-time High-Precision Tracking:** Saat penemu men-scan, sistem meminta izin GPS. Jika diizinkan, pemilik mendapatkan titik koordinat presisi (Google Maps link) via WhatsApp detik itu juga.
* **Double-Sided Customization:** Satu sisi berisi foto kustom (keluarga/anak/peliharaan), sisi lainnya berisi QR Code dengan desain elegan.
* **Priority WhatsApp Notification:** Menggunakan jalur API WhatsApp prioritas agar pesan peringatan sampai dalam hitungan milidetik.
* **Lifetime Scan History:** Akses selamanya ke riwayat scan, lengkap dengan data perangkat penemu (browser, merk HP, dll).
* **Premium Profile Themes:** Halaman publik yang bisa diganti-ganti temanya (misal: Tema Elegan, Tema Darurat, atau Tema Ceria).
* **Verified Gold Badge:** Centang emas di profil publik, menandakan perlindungan tingkat tinggi.

---

## 2. Alur User & Aplikasi (The Custom Journey)

Karena ini produk kustom, alurnya lebih personal:

### **A. Fase Desain (The Experience)**
1.  **Landing Page:** User memilih "Gantungan Akrilik Premium".
2.  **Live Customizer:** * User mengunggah foto.
    * User melakukan *crop/rotate* foto di website (menggunakan `react-easy-crop`).
    * User memilih bentuk akrilik (Bulat/Kotak/Love).
    * User memilih template desain sisi belakang (Warna/Text).
3.  **Checkout:** Pembayaran via **QRIS**.

### **B. Fase Produksi (The Magic)**
1.  **High-Res Generation:** Sistem men-generate file `.pdf` atau `.png` 300 DPI yang menggabungkan Foto User + QR Code unik dalam satu *layout* siap cetak.
2.  **Handmade Assembly:** Admin (Anda) mencetak, memotong akrilik, dan merakitnya dengan *ring* gantungan besi berkualitas tinggi.
3.  **Pre-Activated:** Karena ini kustom, tag ini **sudah otomatis terhubung** ke akun pembeli sejak mereka bayar. Jadi saat barang sampai, user tinggal pakai (tidak perlu aktivasi lagi).

## 3. Desain UI Dashboard Premium

Tampilan dashboard untuk pemilik akrilik harus terasa "mahal":

* **Dark Mode Elegant:** Gunakan aksen warna emas atau biru elektrik.
* **Interactive Map:** Tampilkan peta interaktif (Leaflet/Google Maps) yang menunjukkan riwayat di mana saja kunci mereka pernah di-scan.
* **Device Info:** Tampilkan detail teknis penemu (Contoh: *"Di-scan oleh iPhone 15 Pro di Banjarmasin"*). Ini memberikan rasa kendali penuh bagi pemilik.

## 🛠️ Ringkasan Teknis (Developer Notes)

Untuk fitur **GPS High-Precision**, berikut cuplikan logika yang harus ada di halaman `/p/[slug]`:

```javascript
// Contoh logika meminta izin lokasi pada penemu
function requestLocation() {
  if (navigator.geolocation) {
    navigator.geolocation.getCurrentPosition((position) => {
      const { latitude, longitude } = position.coords;
      // Kirim koordinat ini ke Server Action Anda
      sendLocationToOwner(latitude, longitude);
    });
  }
}
```

Dalam dunia *e-commerce* produk kustom, **Mockup Visual** adalah senjata rahasia untuk mengubah pengunjung yang "cuma lihat-lihat" menjadi pembeli yang berani bayar mahal.

Tanpa mockup, user harus berimajinasi sendiri, dan imajinasi yang ragu-ragu biasanya berakhir dengan pembatalan pesanan.

Berikut adalah alasan teknis dan strategis mengapa Anda harus menyediakan pilihan mockup desain:

---

### 1. Meningkatkan "Perceived Value" (Nilai Jual)
Gantungan kunci akrilik tanpa desain mungkin terlihat seperti barang murahan. Namun, dengan menyediakan mockup bertema (misal: Tema Minimalis, Tema Kids, atau Tema Racing), user merasa mereka sedang membeli **produk desainer**, bukan sekadar kepingan plastik.

### 2. Fitur "Live Preview" (Kepuasan Instan)
Saat user mengunggah foto anak atau kucingnya, dan sistem Anda secara otomatis menampilkan foto tersebut **di dalam** bingkai gantungan kunci yang mengkilap di layar HP mereka, tercipta efek *"Wow"*. Ini adalah pemicu emosional terkuat untuk klik tombol "Beli Sekarang".

### 3. Pilihan Bentuk Fisik (Die-Cut)
Mockup membantu user memilih bentuk fisik yang mereka inginkan. Anda bisa menyediakan 3-4 bentuk standar:
* **Circle (Lingkaran):** Klasik dan aman.
* **Rounded Square (Kotak Tumpul):** Modern dan maskulin.
* **Heart (Hati):** Populer untuk pasangan atau kado.
* **Capsule (Kapsul):** Unik dan terlihat lebih teknis.

### Cara Teknis Implementasi di Next.js (Tanpa Coding Rumit)

Anda tidak perlu software 3D canggih. Gunakan teknik **CSS Overlay**:

1.  **Frame Image:** Siapkan gambar `.png` transparan berbentuk gantungan kunci yang memiliki efek bayangan (*shadow*) dan kilauan (*glossy*) di bagian pinggirnya.
2.  **User Image:** Letakkan foto yang diunggah user di *layer* bawah menggunakan `z-index`.
3.  **CSS Masking:** Gunakan properti `mask-image` di Tailwind agar foto user otomatis terpotong mengikuti bentuk gantungan kunci (lingkaran/kotak).

```html
<div className="relative w-64 h-64">
  {/* Foto User di bawah */}
  <img src={userPhoto} className="absolute inset-0 object-cover mask-circle" />
  
  {/* Frame Gantungan Kunci (Glossy & Ring) di atas */}
  <img src="/frames/acrylic-circle.png" className="absolute inset-0 pointer-events-none" />
</div>
```

Ini akan menarik bagi user yang ingin privasi (tidak mau pakai foto asli) tapi ingin barang yang terlihat estetik.
