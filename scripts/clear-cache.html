🧹 Clear Service Worker & Cache

📋 Instructions:

  1. Click the buttons below to clear service worker and cache
  2. Close all tabs of this application
  3. Restart your development server: npm run dev
  4. Open the application again in a fresh tab

🔍 Manual Steps (if buttons don't work):

  1. Open DevTools (F12 or Right-click → Inspect)
  2. Go to Application tab
  3. Click Service Workers on the left
  4. Click "Unregister" for all service workers
  5. Click Clear storage → "Clear site data"
  6. Close DevTools and refresh the page (Ctrl+Shift+R)
